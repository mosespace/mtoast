## Installation
Install the component from your command line.

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>
  <Tab>```pnpm i @mtoast/mosespace ```</Tab>
  <Tab>```npm i @mtoast/mosespace ```</Tab>
  <Tab>```yarn add @mtoast/mosespace ```</Tab>
  <Tab>```bun add @mtoast/mosespace ```</Tab>

### Add Toaster to your app

It can be placed anywhere, even in server components such as `layout.tsx`.

```tsx
import { Toaster } from '@mtoast/mosespace';

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en">
      <body>
        {children}
        <Toaster />
      </body>
    </html>
  );
}
```

### Render a toast

```tsx
import { toast } from '@mtoast/mosespace';

function MyToast() {
  return (
    <button
      onClick={() => toast.success('Rendered!', 'This is a mtoast toast')}
    >
      Render my toast
    </button>
  );
}
```
