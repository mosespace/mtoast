**1.** Fork [this](https://github.com/mosespace/mtoast) repository.

**2.** Clone your forked copy of the project.

```
git clone https://github.com/<your_user_name>/mtoast.git
```

**3.** Navigate to the project directory.

```
cd mtoast
```

**4.** Add a reference (remote) to the original repository.

```
git remote add upstream https://github.com/mosespace/mtoast.git
```

**5.** Check the remotes for this repository.

```
git remote -v
```

**6.** Always pull from the upstream repository to keep your main branch up to date.

```
git pull upstream main
```

**7.** Create a new branch for your changes.

```
git checkout -b <your_branch_name>
```

**8.** Make your changes to the codebase.

**9.** Stage your changes.

```
git add .
```

**10.** Commit your changes.

```
git commit -m "Describe your changes"
```

**11.** Push your branch to your fork.

```
git push -u origin <your_branch_name>
```

**12.** Open a pull request on GitHub.

**13.** Add a clear title and description to your pull request explaining your changes.

**14.** Click on `Create Pull Request`.
